Задачи

1. Брендинг и визуальная идентификация

UX/UI дизайнер

  1. Создать логотип:
    • Исследовать референсы для лого (2 ч).
    • Нарисовать 3 варианта логотипа (4 ч).
    • Согласовать лучший вариант с командой (1 ч).
  1. Разработать цветовую палитру:
    • Подобрать основные и дополнительные цвета (2 ч).
  1. Создать шрифтовую пару:
    • Подобрать шрифты для интерфейса и заголовков (2 ч).
  1. Разработать UI-кит:
    • Создать кнопки, карточки, иконки (4 ч).

2. Пользовательские сценарии и структура

Product Manager

  1. Составить список ключевых действий пользователя:
    • Определить шаги: тестирование, выбор рекомендаций, диалог с ботом (3 ч).
  1. Написать сценарии для взаимодействия:
    • Описать пошаговые сценарии для каждого действия (6 ч).
  1. Составить текстовые шаблоны для бота:
    • Приветствие, объяснение теста, вывод рекомендаций (6 ч).

3. Прототипирование

UX/UI дизайнер

  1. Создать прототип интерфейса бота:
    • Нарисовать экран приветствия и теста (4 ч).
    • Нарисовать экран выбора рекомендаций (4 ч).
    • Нарисовать экран просмотра карточек ресурсов (4 ч).
  1. Создать прототип сайта-визитки:
    • Разработать макет главной страницы (5 ч).
    • Нарисовать страницы "О нас" и "Контакты" (5 ч).

4. Разработка базы данных

Backend-разработчик

  1. Составить модель базы данных:
    • Определить таблицы (объекты, теги, пользователи) (4 ч).
  1. Настроить базу данных:
    • Создать таблицы в SQLite/PostgreSQL (6 ч).
  1. Наполнить базу тестовыми данными:
    • Добавить 20-30 записей для проверки (3 ч).

5. Кодовая база для бота

Backend-разработчик

  1. Настроить серверное окружение:
    • Развернуть сервер на Heroku/Ubuntu (3 ч).
    • Установить фреймворк для бота (Telegraf, Flask) (2 ч).
  1. Написать код для базового функционала:
    • Реализовать приветствие и вводную часть (4 ч).
    • Настроить обработку текстовых сообщений (4 ч).
    • Реализовать кнопки и меню (6 ч).
  1. Настроить API для рекомендаций:
    • Подключить внешний API (например, Spoonacular) (6 ч).

6. Разработка сайта

Frontend-разработчик

  1. Верстка страницы:
    • Сверстать главную страницу сайта (5 ч).
    • Добавить страницу "Контакты" (3 ч).
  1. Интеграция с backend:
    • Подключить базу данных через API (6 ч).

7. Тестирование

QA-инженер

  1. Написать чек-лист для тестирования:
    • Проверить функциональность бота (3 ч).
    • Проверить навигацию сайта (3 ч).
  1. Провести тестирование:
    • Тестировать сценарии взаимодействия с ботом (6 ч).
    • Тестировать сайт (4 ч).

8. Подготовка к запуску

PM, Маркетолог

  1. Подготовить презентацию:
    • Составить текст и визуализацию (6 ч).
  1. Настроить партнёрские программы:
    • Зарегистрировать affiliate links (5 ч).
